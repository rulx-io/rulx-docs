---

Flows are the main orchestration artifact in Rulx.

From the build surface you can:

- browse the flow registry
- create a new flow draft
- edit flow metadata and callable interface
- open the visual editor
- attach tests and trigger bindings
- inspect per-flow run history

## Main flow pages

The app exposes a consistent set of flow pages:

- `general`
- `config`
- `editor`
- `tests`
- `runs`
- `registry`
