> Hinweis zu Pre-Alpha: Lokale Entwicklung ist heute der am besten unterstützte Deployment-Pfad.

## Hauptbefehl

Vom Monorepo-Root aus:

```bash
cp .env.example .env.local
pnpm dev
```

## Was startet

Der lokale Stack startet:

- Controller
- Web
- Node-Worker
- Runner
- Gateway
