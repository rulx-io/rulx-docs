Die flow-spezifische Runs-Seite verengt die workspace-weite Run-Oberfläche auf ein einzelnes Artefakt.

Nutze sie für Fragen wie:

- Wie verhält sich dieser Flow über die Zeit?
- Hat eine letzte Änderung die Ausführung verschlechtert?
- Welchen Run sollte ich aus dem Editor-Kontext im Detail öffnen?
