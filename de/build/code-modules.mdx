---
title: 'Code-Module'
description: 'Custom Nodes, die als Workspace-Artefakte verwaltet werden'
---

Code-Module sind die Custom-Node-Oberfläche in Rulx.

Sie erlauben dir, wiederverwendbares Verhalten als Workspace-verwaltete Artefakte zu paketieren, auf die Flows aufbauen können.

## Was sie unterscheidet

Im Vergleich zu Flows und Schemas haben Code-Module einen stärkeren Fokus auf Dependency-Management.

Das zeigt sich in:

- Registry-Seiten der Module
- generiertem Source-Code im Editor
- Installationsjobs für Node-Dependencies
- Workspace-Package-Registry-Konfiguration
