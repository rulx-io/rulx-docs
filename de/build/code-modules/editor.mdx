Im Code-Modul-Editor wird das Verhalten von Custom Nodes erstellt.

## Zentrale Ideen

Der Editor kombiniert:

- eine visuelle Authoring-Oberfläche
- generierten Source-Code
- typisierte Ports und schema-bewusste Bindings
- importierten Flow- und Workspace-Kontext, wo nötig
