Code-Module können Runtime-Dependencies benötigen, die explizit installiert und verwaltet werden müssen.

## Runtime-Dependency-Oberflächen

Controller und Build-UI stellen dafür Oberflächen bereit wie:

- Installationsjobs für Artefakt-Drafts und -Versionen
- Workspace-Package-Registries
- Bulk-Bump-Flows für Dependencies
