This group covers the execution history and debugging side of the controller.

## Job endpoints

- `GET /v1/w/{workspaceId}/jobs`
- `POST /v1/w/{workspaceId}/jobs`
- `GET /v1/w/{workspaceId}/jobs/{jobId}`
- `POST /v1/w/{workspaceId}/jobs/{jobId}/cancel`

## Debugging endpoints

- `GET /v1/w/{workspaceId}/jobs/{jobId}/logs`
- `GET /v1/w/{workspaceId}/jobs/{jobId}/logs/stream`
- `GET /v1/w/{workspaceId}/jobs/{jobId}/timeline`
