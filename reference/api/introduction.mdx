The controller API is the public HTTP surface for operating Rulx.

## Versioning

The current snapshot is versioned under `/v1/...`.

The checked-in contract snapshot lives at:

- [OpenAPI snapshot](/reference/api/openapi.json)

## What it covers today

The current snapshot includes surfaces for:

- workspaces
- artifacts, drafts, versions, and publish flows
- jobs, logs, and timeline
- runners and services
- trigger bindings
- inputs and node catalog
- node dependency workflows
