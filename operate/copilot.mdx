Rulx includes assistant surfaces directly inside the workspace UI.

## What exists today

The current experience combines:

- a search and action palette
- a route-aware workspace chat drawer
- current-context hints tied to the active workspace, page, flow, or run

## What it is good at

The current assistant is shaped around:

- searching routes and surfaces
- summarizing the current build or run context
- helping you stay oriented while moving between the editor, runs, and settings

## Current limits

Treat the assistant today as:

- a navigation and explanation surface
- a persistent planning surface for the current route and workspace

not as a complete automation layer for every artifact workflow.
